## 📂 Estrutura do projeto

## 🏗️ Sobre o projeto

* Implementado em **Java (Spring MVC), HTML, CSS e JavaScript**
* Objetivo: praticar conceitos de **front-end, back-end e integração web**
* A versão final (**Atividade 13**) consolida todas as funcionalidades anteriores

---

## 📌 Dicas de uso

* Use a **Atividade 13** para testes e execução, pois é a versão final.
* Tags são **fixas**, se quiser alterar uma versão, crie uma branch:

```bash
git checkout -b minha-branch
```

* O histórico das tags permite acompanhar a evolução do projeto de forma prática.
